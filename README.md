# Ganzhi2

NSUserTrackingUsageDescription는 iOS 앱이 사용자의 데이터(활동 기록)를 추적하여 광고나 분석 목적으로 활용하려 할 때, 사용자에게 "왜 추적 허용이 필요한지" 설명하는 문구를 정의하는 항목입니다.

## 현재 Ganzhi 앱의 

Info.plist
에는 다음과 같이 설정되어 있습니다:

문구: "귀하의 취향에 맞는 사주 분석과 맞춤형 광고를 제공하기 위해 활동을 추적합니다."
이 설정이 하는 구체적인 역할은 다음과 같습니다:

1. 개인정보 보호 투명성 (ATT) 준수
Apple은 iOS 14.5부터 App Tracking Transparency (ATT) 정책을 시행하고 있습니다. 앱이 사용자의 기기 식별자(IDFA)에 접근하여 다른 앱이나 웹사이트에서의 활동을 추적하려면 반드시 사용자의 명시적인 동의를 받아야 합니다.

2. 권한 요청 팝업의 본문 표시
앱이 실행될 때(혹은 광고 로드 전) 뜨는 **"앱이 다른 회사의 앱 및 웹사이트에 걸친 사용자의 활동을 추적하도록 허용하시겠습니까?"**라는 팝업 창의 본문 내용으로 이 문구가 사용됩니다.

3. 맞춤형 광고(고단가 광고) 노출
사용자가 '허용'을 누르면 Google AdMob과 같은 광고 네트워크가 사용자의 관심사를 파악할 수 있게 됩니다.

결과: 사용자에게는 더 관련성 높은 광고가 보이고, 앱 개발자에게는 단가가 더 높은 맞춤형 광고가 송출되어 수익 향상에 도움이 됩니다.
요약
NSUserTrackingUsageDescription은 **앱 광고 수익 최적화와 개인정보 보호 정책 준수를 위해 필수적인 '설득 문구'**라고 이해하시면 됩니다. :)

## Ganzhi(간지) 앱이 수집하고 활용하는 정보는 크게 두 종류로 나눌 수 있습니다.

1. 사용자가 직접 입력하는 사주 정보 (로컬 저장)
앱의 핵심 기능인 사주 분석을 위해 사용되는 정보입니다. 이 정보들은 **사용자의 기기에만 저장(@AppStorage)**되며, 개발자가 별도로 수집하거나 서버에 쌓아두지 않습니다.

생년월일시: 년, 월, 일, 시, 분 (사주 팔자 도출용)
성별: 남성/여성 (명리학적 대운 및 해석 보정용)
양력/음력 여부: 달력 계산용
2. 서비스 제공을 위해 외부로 전달되는 정보 (AI 및 광고)
앱 내 기능을 실행하는 과정에서 외부 서비스(OpenAI, Google AdMob)에 일시적으로 전달되는 정보입니다.

AI 분석용 데이터: 심층 분석 버튼을 누를 때, 위에서 입력한 사주(간지)와 성별, 사용자의 특별 요청 사항이 사주 풀이를 위해 OpenAI API로 전달됩니다. (개인 식별 정보는 포함되지 않습니다.)
광고 및 분석용 데이터 (ATT 허용 시): 사용자가 추적 허용을 선택한 경우, **기기 식별자(IDFA)**가 Google AdMob에 전달되어 맞춤형 광고를 제공하는 데 사용됩니다.
🔐 보안 및 개인정보 관리 특징
서버 저장 없음: 별도의 회원가입이나 로그인 절차가 없으므로, 사용자의 이름, 이메일, 전화번호와 같은 민감한 개인 식별 정보는 절대 수집하지 않습니다.
로컬 관리: 사용자가 '내 정보로 저장'을 선택한 데이터는 본인의 스마트폰 내부 저장소에만 안전하게 보관되며, 앱을 삭제하면 함께 제거됩니다.
요약하자면, Ganzhi 앱은 정확한 사주 풀이를 위한 최소한의 역학 데이터만 활용하며, 이를 외부에 무분별하게 수집하거나 활용하지 않는 개인정보 보호 친화적인 앱입니다. :)
